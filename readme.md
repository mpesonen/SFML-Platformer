# SFML-Platformer

Simple test project for [SFML](https://github.com/SFML/SFML), modern C++ and CMake.

https://user-images.githubusercontent.com/2873991/235761073-3d47f484-1617-4b0f-af5c-febfc3db22b0.mov

## Building instructions

mkdir build
cd build
cmake ..
make install
./sfml-platformer
